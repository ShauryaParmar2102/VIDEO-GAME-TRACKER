# 🕹️ Video Game Tracker

A full-stack web application to track your video game collection. Users can add, delete, search, and filter games by status (Played/Unplayed) and track details like company, description, and date added — all built with ** HTML, CSS, JavaScript, Node.js, and Express**.

---

## ⚡ Features

- ➕ Add new games with **Title**, **Company**, and **Description**  
- 🟢/🔴 Track if games are **Played** or **Unplayed**  
- ❌ Delete games from your collection  
- 🔍 Search for games by title or company  
- 📅 Automatically adds a **timestamp** when games are added

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, Vanilla JavaScript
- **Backend:** Node.js, Express  
- **Data Storage:** JSON file (`games.json`)  
- **Communication:** RESTful API using `fetch()`

---

## 🚀 Getting Started

1. **Download the project**  
   - Press the **green Code button** on GitHub  
   - Click **Download ZIP**  
   - **Unzip** the files to a folder on your computer  

2. **Open the project folder** in **Visual Studio Code**  

**Install Node Modules (THIS IS NEEDED FOR PROJECT TO RUN)**

How to Run the Project:

1: Open the terminal in your project folder.

2: Install the dependencies (this will create the node_modules folder):

3: npm install

4: Start the server:

5: npm run start

6: Open your browser and go to http://localhost:5000
